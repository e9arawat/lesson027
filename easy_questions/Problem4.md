## Number Addition.

Example:  

                    Input: [12, 15, 3, 10]
                    Output: 40
                
                    Input: [17, 5, 3, 5]
                    Output: 30

Write a function **add_list(input_list: list)**, *where **input_list** contains **n** numbers each having **m** digits in it*, in solution.py which will return the sum of all the numbers in the input_list.
